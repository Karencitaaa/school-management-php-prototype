# School Management CRUD System

This project is a basic School Management System developed using PHP.  
It allows administrators to manage core academic entities through CRUD operations.

## Features
- Create, edit, view, and delete students
- Manage teachers, courses, and academic careers
- Simple and clear PHP structure
- Shared components using header and footer files
- Docker configuration for easy local setup

## Technologies Used
- PHP
- MySQL
- Docker & Docker Compose
- HTML / CSS

## Project Purpose
This project was created as an early-stage prototype to understand backend development fundamentals, database interactions, and CRUD operations before migrating to a more structured MVC approach.

## How to Run (Optional)
1. Clone the repository
2. Run `docker-compose up`
3. Access the app via `http://localhost`

## Notes
This repository represents an initial version of the project before transitioning to a full MVC architecture.
